# PowerShell Commands Cheatsheet

| Command               | Description                                                                               | Example                                                                | Documentation                                                                                                               |
| --------------------- | ----------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| `Get-Command`         | Gets all commands                                                                         | `Get-Command`                                                          | [Get-Command](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/get-command)                     |
| `Get-Help`            | Displays help information for commands                                                    | `Get-Help Get-Command`                                                 | [Get-Help](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/get-help)                           |
| `Get-Process`         | Gets the processes that are running on the local computer                                 | `Get-Process`                                                          | [Get-Process](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-process)               |
| `Get-Service`         | Gets the status of services on a local or remote computer                                 | `Get-Service`                                                          | [Get-Service](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-service)               |
| `Start-Service`       | Starts a stopped service                                                                  | `Start-Service -Name "wuauserv"`                                       | [Start-Service](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/start-service)           |
| `Stop-Service`        | Stops a running service                                                                   | `Stop-Service -Name "wuauserv"`                                        | [Stop-Service](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/stop-service)             |
| `Restart-Service`     | Restarts a service                                                                        | `Restart-Service -Name "wuauserv"`                                     | [Restart-Service](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/restart-service)       |
| `Set-ExecutionPolicy` | Changes the user preference for the PowerShell script execution policy                    | `Set-ExecutionPolicy RemoteSigned`                                     | [Set-ExecutionPolicy](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.security/set-executionpolicy) |
| `Get-EventLog`        | Gets the events in the event log on the local or remote computers                         | `Get-EventLog -LogName "System"`                                       | [Get-EventLog](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-eventlog)             |
| `New-Item`            | Creates a new item (file, directory, registry key, etc.)                                  | `New-Item -Path "C:\temp\newfile.txt" -ItemType "File"`                | [New-Item](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/new-item)                     |
| `Remove-Item`         | Deletes files and directories                                                             | `Remove-Item -Path "C:\temp\newfile.txt"`                              | [Remove-Item](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/remove-item)               |
| `Copy-Item`           | Copies an item from one location to another                                               | `Copy-Item -Path "C:\temp\file.txt" -Destination "C:\backup\file.txt"` | [Copy-Item](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/copy-item)                   |
| `Move-Item`           | Moves an item from one location to another                                                | `Move-Item -Path "C:\temp\file.txt" -Destination "C:\backup\file.txt"` | [Move-Item](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/move-item)                   |
| `Rename-Item`         | Renames an item                                                                           | `Rename-Item -Path "C:\temp\file.txt" -NewName "newfile.txt"`          | [Rename-Item](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/rename-item)               |
| `Get-Content`         | Gets the content of the item at the specified location                                    | `Get-Content -Path "C:\temp\file.txt"`                                 | [Get-Content](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-content)               |
| `Set-Content`         | Writes or replaces the content in an item with new content                                | `Set-Content -Path "C:\temp\file.txt" -Value "New content"`            | [Set-Content](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/set-content)               |
| `Add-Content`         | Appends content to the specified items                                                    | `Add-Content -Path "C:\temp\file.txt" -Value "Additional content"`     | [Add-Content](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/add-content)               |
| `Clear-Content`       | Deletes the contents of an item, but does not delete the item                             | `Clear-Content -Path "C:\temp\file.txt"`                               | [Clear-Content](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/clear-content)           |
| `Get-ChildItem`       | Gets the items and child items in one or more specified locations                         | `Get-ChildItem -Path "C:\temp"`                                        | [Get-ChildItem](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-childitem)           |
| `Get-Location`        | Gets information about the current working location or a location stack                   | `Get-Location`                                                         | [Get-Location](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-location)             |
| `Set-Location`        | Sets the current working location to a specified location                                 | `Set-Location -Path "C:\temp"`                                         | [Set-Location](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/set-location)             |
| `Invoke-Command`      | Runs commands on local and remote computers                                               | `Invoke-Command -ScriptBlock { Get-Process }`                          | [Invoke-Command](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/invoke-command)               |
| `Import-Module`       | Adds one or more modules to the current session                                           | `Import-Module -Name "ActiveDirectory"`                                | [Import-Module](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/import-module)                 |
| `Export-ModuleMember` | Specifies the module members that are exported                                            | `Export-ModuleMember -Function "Get-MyFunction"`                       | [Export-ModuleMember](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/export-modulemember)     |
| `Get-Module`          | Gets the modules that have been imported or that can be imported into the current session | `Get-Module -ListAvailable`                                            | [Get-Module](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/get-module)                       |
| `Remove-Module`       | Removes modules from the current session                                                  | `Remove-Module -Name "ActiveDirectory"`                                | [Remove-Module](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/remove-module)                 |
| `Get-Variable`        | Gets the PowerShell variables in the current console                                      | `Get-Variable`                                                         | [Get-Variable](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.utility/get-variable)                |
| `Set-Variable`        | Sets the value of a variable                                                              | `Set-Variable -Name "MyVar" -Value "MyValue"`                          | [Set-Variable](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.utility/set-variable)                |
| `Remove-Variable`     | Deletes a variable and its value                                                          | `Remove-Variable -Name "MyVar"`                                        | [Remove-Variable](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.utility/remove-variable)          |
| `Write-Output`        | Sends the specified objects to the next command in the pipeline                           | `Write-Output "Hello, World!"`                                         | [Write-Output](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/write-output)                   |
| `Write-Host`          | Displays objects in the host program                                                      | `Write-Host "Hello, World!"`                                           | [Write-Host](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.utility/write-host)                    |
| `Read-Host`           | Reads a line of input from the console                                                    | `$input = Read-Host "Enter your name"`                                 | [Read-Host](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.utility/read-host)                      |
| `Start-Job`           | Starts a PowerShell background job                                                        | `Start-Job -ScriptBlock { Get-Process }`                               | [Start-Job](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/start-job)                         |
| `Get-Job`             | Gets the Windows PowerShell background jobs that are running in the current session       | `Get-Job`                                                              | [Get-Job](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/get-job)                             |
| `Stop-Job`            | Stops a PowerShell background job                                                         | `Stop-Job -Id 1`                                                       | [Stop-Job](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/stop-job)                           |
| `Receive-Job`         | Gets the results of a PowerShell background job                                           | `Receive-Job -Id 1`                                                    | [Receive-Job](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/receive-job)                     |
| `Remove-Job`          | Deletes a PowerShell background job                                                       | `Remove-Job -Id 1`                                                     | [Remove-Job](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/remove-job)                       |
